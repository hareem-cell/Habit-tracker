# Habit Tracker

A weekly habit tracker web app built using Flask.

 **Features**
- Add habits
- Edit habits
- Delete habits
- Weekly grid tracking
- Data persistence using JSON

**How to Run Locally**

**Install dependencies**
pip install -r requirements.txt (tells python .. read this file .. install Flask .. install gunicorn)
**Run the app**
python app.py
Then open:
http://127.0.0.1:5000
**Tech Stack**
- Flask
- HTML
- CSS
- JSON storage
